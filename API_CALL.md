# VOICEVOX API 调用手册（通用模板）

> 更新时间：2026-02-11 03:11:19（本机实时从 `/voices` 拉取）

## 1. 账号与鉴权
- API Key：`YOUR_API_KEY`
- 额度：由服务端配置决定
- 基础地址：`https://YOUR_DOMAIN`
- 鉴权头：`X-API-Key: YOUR_API_KEY`

## 2. 接口总览
- `GET /voices`：获取角色与声线清单（含 `speaker` 编号）
- `GET /check_key?key=...`：查询 key 是否可用与额度
- `POST /tts`：JSON 合成（不上传文件）
- `POST /tts_custom`：`multipart/form-data` 合成（可上传自定义 BGM）
- `GET /character_info?uuid=...`：获取角色头像与试听音频 URL

## 3. 标准调用（/tts）
```bash
curl -sS -X POST 'https://YOUR_DOMAIN/tts' \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_API_KEY' \
  --data '{
    "text": "中国有句古话，识时务者为俊杰。",
    "speaker": 22,
    "speedScale": 1.2,
    "pitchScale": 0.0,
    "intonationScale": 1.1,
    "volumeScale": 1.0,
    "prePhonemeLength": 0.1,
    "postPhonemeLength": 0.1,
    "outputSamplingRate": 48000,
    "outputStereo": true,
    "kana": "",
    "bgmEnabled": true,
    "bgmVolume": 0.35
  }' \
  -o out.wav
```

## 4. 自定义 BGM 调用（/tts_custom）
```bash
curl -sS -X POST 'https://YOUR_DOMAIN/tts_custom' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -F 'text=中国有句古话，识时务者为俊杰。' \
  -F 'speaker=22' \
  -F 'mode=pseudo_jp' \
  -F 'speedScale=1.2' \
  -F 'pitchScale=0.0' \
  -F 'intonationScale=1.1' \
  -F 'volumeScale=1.0' \
  -F 'prePhonemeLength=0.1' \
  -F 'postPhonemeLength=0.1' \
  -F 'outputSamplingRate=48000' \
  -F 'outputStereo=true' \
  -F 'kana=' \
  -F 'bgmEnabled=true' \
  -F 'bgmVolume=0.35' \
  -F 'bgmFile=@/absolute/path/to/your_bgm.mp3' \
  -o out_custom_bgm.wav
```

## 5. 参数说明与建议区间
| 参数 | 说明 | 建议 |
|---|---|---|
| `speaker` | 声线编号 | 从本文第 8 节查表 |
| `speedScale` | 语速 | `0.8 ~ 1.4` |
| `pitchScale` | 音高 | `-0.15 ~ 0.15` |
| `intonationScale` | 语调起伏 | `0.8 ~ 1.3` |
| `volumeScale` | 人声音量 | `0.8 ~ 1.4` |
| `prePhonemeLength` | 前停顿 | `0.0 ~ 0.3` |
| `postPhonemeLength` | 后停顿 | `0.0 ~ 0.3` |
| `outputSamplingRate` | 采样率 | 推荐 `48000` |
| `outputStereo` | 立体声 | 推荐 `true` |
| `bgmEnabled` | 默认 BGM 开关 | `true/false` |
| `bgmVolume` | BGM 音量 | 推荐 `0.2 ~ 0.45` |

## 6. 俊达萌耳语推荐配置
- 低语（`speaker=22`）：更稳，适合一般耳语内容
- 悄悄话（`speaker=38`）：更贴耳，适合 ASMR/私语
- 推荐模板：`speedScale=1.15~1.25`、`intonationScale=1.0~1.15`、`bgmVolume=0.25~0.4`

## 7. 常见错误与排障
- `401 Invalid key or no credits`：API Key 错误或额度不足
- `402 Insufficient balance`：用户余额不足
- `500/502`：后端或语音引擎瞬时异常，建议重试
- 返回音频不符合预期：先固定 `speaker`，再每次只调一个参数做 A/B 测试
- 合成慢：优先缩短文本、减小段落，避免一次超长句

## 9. 建议给第三方的最小接入流程
1. 先调 `GET /voices` 拉取最新 `speaker` 编号。
2. 在业务侧持久化“角色名-声线名-speaker”。
3. 生成时仅透出少量可调参数（语速/语调/BGM 音量），避免误调。
4. 失败时记录完整请求参数（不要记录明文密钥）。

## 10. 一键刷新清单命令
当引擎升级后，声线可能变化。先执行：
```bash
curl -sS 'https://YOUR_DOMAIN/voices' | jq
```
并以最新输出为准更新对照表。
